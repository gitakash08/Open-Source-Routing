About the App

Our Map App is an interactive mapping application that allows you to explore and find routes between locations on a map.

You can set source and destination markers on the map by clicking on it. The app will then calculate and display the route between these markers, making it easy to plan your journeys.

Key Features

📍
Interactive Map with Open Street Map Tiles
📍
Set Source and Destination Markers
📍
Multiple Routes with Different Color Geometry
📍
Find Routes Between Locations
📍
User-Friendly Interface
